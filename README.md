# Date Quest ❤️

Date Quest is a playful and romantic single-page web application built with Vue 3, TypeScript, Tailwind CSS, and Lucide Vue Next. The experience guides visitors through a charming interactive journey, starting with the classic "Will you go on a date with me?" question and ending with a personalized date invitation summary.

The application combines smooth animations, floating hearts, playful interactions, and elegant card-based layouts to create a memorable experience. Users can choose their preferred date, pickup time, food vibe, and excitement level before receiving a customized summary complete with a generated QR code and downloadable invitation card.

## Features

* Romantic and playful user experience
* Interactive "Yes" and evasive "No" button mechanics
* Multi-step date planning journey
* Food vibe selection with visual cards
* Animated excitement meter
* Personalized invitation summary
* Client-side QR code generation
* Downloadable invitation image
* Mobile-first responsive design
* Smooth transitions and micro-animations
* Floating hearts and celebration effects
* No backend required
* Fully client-side SPA architecture

## Tech Stack

* Vue 3
* TypeScript
* Tailwind CSS
* Lucide Vue Next

## Architecture

* Single Page Application (SPA)
* No SSR
* No SEO requirements
* No database
* No authentication
* Local state management only

Date Quest focuses on creating a fun, memorable, and shareable romantic experience through delightful interactions and modern frontend design.
